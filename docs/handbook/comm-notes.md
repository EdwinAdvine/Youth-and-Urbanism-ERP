# Notes: Rich Text Notes & Notebooks

The Notes module is your team's shared writing space. Use it for meeting minutes, internal SOPs, product briefs, customer research, or any structured content that needs more than a chat message but less than a full document.

## Creating a Note

1. Open the **Notes** module from the left sidebar.
2. Click **New Note**.
3. Give the note a **title** and start writing in the body — the editor is powered by Tiptap, a modern rich text engine.

## Rich Text Formatting

The Tiptap editor supports a full range of formatting options:

- **Headings** (H1–H3) for structuring sections
- **Bold**, *italic*, and underline for emphasis
- **Bullet lists** and **numbered lists**
- **Task lists** (checkboxes) for lightweight action items within a note
- **Code blocks** with syntax highlighting — useful for technical runbooks or API notes
- **Tables** for structured data such as pricing comparisons or meeting agendas
- **Inline images** dragged in or pasted from the clipboard
- **Embeds** — paste a URL to embed a Drive file preview or a link card

Use `/` (slash command) anywhere in the body to quickly insert any of these blocks without reaching for the toolbar.

## Organising Notes into Notebooks

Notebooks work like folders. Create notebooks for different contexts — for example: "Sales Team", "HR Policies", "Project Phoenix", "Customer Onboarding".

1. In the Notes sidebar, click **New Notebook**.
2. Name the notebook and optionally set a colour.
3. Drag existing notes into the notebook, or select the notebook when creating a new note.

Notes can be nested: a notebook can contain sub-notebooks, giving you as much hierarchy as your team needs.

## Pinning Important Notes

Pin a note to keep it at the top of your list. Open any note and click the **Pin** icon (top-right). Pinned notes appear in a dedicated **Pinned** section above your notebook list — useful for SOPs, contact directories, or any reference material you access constantly.

## Sharing and Collaboration

By default, a note is private to its creator. To share:

1. Open the note and click **Share**.
2. Search for team members by name and set their access level: **View** or **Edit**.
3. Click **Done** — the recipients see the note appear in their **Shared with Me** section.

**Real-time collaboration** is supported: multiple users can edit the same note simultaneously. Each collaborator's cursor is shown in a different colour with their name, similar to Google Docs. Changes are auto-saved continuously.

## Linking Notes to Other Modules

Notes become even more powerful when linked to records elsewhere in Urban Vibes Dynamics:

- **CRM Contact or Deal**: open a contact/deal record, go to the **Notes** tab, and click **Attach Existing Note** or **Create Note** — the note appears both in Notes and on the CRM record.
- **Project**: link a note to a project from the Project detail page for meeting minutes, specs, or retros.
- **Meeting**: when a Jitsi meeting ends, notes are auto-created and linked to the calendar event (see the Jitsi article).

All linked notes are visible from both the Notes module and the source record, so context is never lost.
