# Project Cross-Module Integrations

Urban Vibes Dynamics's projects module is not an island. Every project can be connected to the deals you are pursuing, the costs you are incurring, the files you are sharing, the meetings you are holding, and the emails your client sends. This article covers how to activate each integration and what you get out of it.

## Linking a Project to a CRM Deal

When a project is won from a sales opportunity, linking it to the CRM deal keeps the commercial context visible throughout delivery.

1. Open the project and go to the **Integrations** tab.
2. Under **CRM Deal**, click **Link Deal**.
3. Search for the deal by client name or deal title and select it.

**What you gain:**

- The deal's value (KES) appears on the project dashboard alongside the budget — so the team can see the revenue context of what they are building.
- The project appears on the CRM deal's timeline, so Sales can track delivery progress without asking the Project Manager.
- When the project is marked Completed, the linked deal can be automatically transitioned to "Closed Won" (configurable in CRM settings).

## Linking Tasks to Finance Expenses

Project costs often live in Finance as expense records. Linking them to tasks keeps the budget burn accurate.

1. Open a task and click **Link Expense** (in the task detail panel, under the Integrations section).
2. Search for an existing expense record in Finance (e.g., "Westgate Electricals — wiring materials KES 45,000") or click **Create Expense** to open a new expense form pre-filled with the project name.
3. Confirm the link.

**What you gain:**

- The expense amount is added to the project's budget burn total alongside logged time costs.
- Finance gets the project code on the expense, making project-level P&L reports possible.
- The task shows the linked cost inline — useful when reviewing whether the scope matches the spend.

## Auto-Creating a Drive Folder for a Project

Every project generates documents: proposals, contracts, meeting notes, deliverables. Urban Vibes Dynamics can automatically create a dedicated Drive folder so files are organised from day one.

1. When creating a new project, toggle **Create Drive Folder** (on the New Project form).
2. Alternatively, for an existing project, go to **Integrations → Drive** and click **Create Project Folder**.

A folder named after the project is created in Drive under a top-level "Projects" folder. The project team are automatically granted access. Any files attached to tasks within the project are stored in this folder.

**What you gain:**

- One-click access to all project files from the project page (Drive tab).
- Files are discoverable in Drive by anyone with access — no hunting through email attachments.
- ONLYOFFICE documents created in the folder can be linked directly to tasks.

## Meetings Linked to Projects

When a meeting is held in the Meetings module (or a Jitsi call is started from within a project), it can be linked to the project.

1. When creating or editing a meeting, select the linked **Project** from the dropdown.
2. After the meeting ends, Urban Vibes Dynamics prompts the host to attach the meeting notes to the project.

**What you gain:**

- All meetings related to the project appear in the project's **Timeline** view in chronological order alongside task completions and milestones.
- Meeting action items (created in the Meetings module) are automatically added as tasks to the project backlog, with the attendee who was assigned the action pre-populated as the task assignee.
- The client (linked via CRM) can be sent an automated meeting summary email from the Meetings module.

## Email Integration: Logging Client Emails on a Project

When a client sends an email about the project to your team's work address, that thread can be pinned to the project:

1. Open the email in the Mail module.
2. Click **Link to Project** (in the email action bar).
3. Select the project.

**What you gain:**

- The email thread appears on the project's **Communications** tab — full context without switching between Mail and Projects.
- Any team member on the project can read the client's email even if they were not originally CC'd.
- Replies sent from within Projects are logged back to the Mail module automatically.

> **Pro Tip:** The most powerful combination is CRM Deal + Drive Folder + Meeting Links. From the moment a deal is created in CRM, link it to a project, auto-create the Drive folder, and ensure all client meetings are linked. By the time you invoice the client, you have a complete project record: scope agreed (CRM), files delivered (Drive), meetings held (Timeline), and costs incurred (Finance expenses) — all in one place.
