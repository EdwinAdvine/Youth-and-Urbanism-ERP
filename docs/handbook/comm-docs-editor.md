# Document Editor (ONLYOFFICE)

The Docs module gives you a full-featured office suite inside Urban Vibes Dynamics, powered by ONLYOFFICE Document Server. Create and edit Word-compatible documents, Excel-compatible spreadsheets, and PowerPoint-compatible presentations — all in your browser, all saved to your Drive, with no Microsoft Office licence required.

## Creating a New Document

1. Open the **Docs** module from the left sidebar.
2. Click **New** and choose the type:
   - **Document** — a Word-compatible `.docx` file
   - **Spreadsheet** — an Excel-compatible `.xlsx` file
   - **Presentation** — a PowerPoint-compatible `.pptx` file
3. Give it a name and click **Create**. The ONLYOFFICE editor opens immediately in your browser.

Alternatively, open the **Drive** module, navigate to a folder, and click **New → Document/Spreadsheet/Presentation** to create the file directly in that folder.

## The Editor

The ONLYOFFICE editor provides the full range of formatting tools you would expect: fonts, styles, paragraph formatting, tables, charts, headers and footers, formulas (in spreadsheets), and slide layouts (in presentations). The toolbar is familiar to anyone who has used Microsoft Office or Google Docs.

Files are auto-saved every few seconds. You can also press **Ctrl+S** (or **Cmd+S** on Mac) to save manually.

## Real-Time Collaborative Editing

Multiple team members can open and edit the same document simultaneously. Each collaborator's cursor is shown in a distinct colour with their name. Changes made by others appear on your screen in real time without needing to refresh.

This makes Docs ideal for collaborative work: writing a proposal together, reviewing a budget spreadsheet as a team, or co-authoring a client presentation.

## Comments and Track Changes

- **Comments**: select any text, right-click, and choose **Add Comment** to leave a note for a collaborator. Comments appear in the right margin. Teammates can reply and resolve comments.
- **Track Changes**: enable from the **Collaboration** menu. Every edit is recorded with the author's name and timestamp, shown in a different colour. Reviewers can accept or reject individual changes — useful for controlled document review workflows (e.g., contracts, HR policies).

## Save as PDF

To export a finished document as a PDF:

1. Click **File → Save as PDF** (or **Download as → PDF**).
2. The PDF is generated and downloaded to your computer, and a copy is saved to Drive alongside the source file.

This is particularly useful for generating clean, non-editable versions of quotations, LPOs, or reports to send to clients.

## Version History

Every saved state of a document is stored as a version. To view history:

1. Click **File → Version History**.
2. A list of versions with timestamps and author names appears in a side panel.
3. Click any version to preview it, and click **Restore** to roll back.

## AI Copilot in the Editor

The Urban Bad AI assistant is available inside the document editor. Select any block of text and click the **AI** button in the toolbar (or press **Ctrl+Shift+Space**) to:

- **Suggest edits** — rewrite the selected text for clarity or tone
- **Summarise** — condense a long section into a short summary
- **Expand** — elaborate on a brief bullet into a full paragraph
- **Translate** — translate selected text to Swahili or English

AI actions are applied as tracked changes so you can review and approve them before accepting.

## Sharing Documents

Click **Share** (top-right) to share the document with specific team members (View or Edit access) or generate a read-only link for external recipients. External links work in any browser — no Urban Vibes Dynamics account required.
